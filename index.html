HTML Crash Kurs

Die wichtigsten HTML-Elemente und Features auf einen Blick

1. Grundstruktur

<!DOCTYPE html>
<html lang="de">
  <head>
    <meta charset="UTF-8">
    <title>Meine Webseite</title>
  </head>
  <body>
    Inhalt
  </body>
</html>

2. Überschriften & Text

  • <h1>...</h1> bis <h6>...</h6>: Überschriften
  • <p>...</p>: Absatz
  • <strong>: Fett <em>: Kursiv
  • <br>: Zeilenumbruch
<h1>Hallo Welt!</h1>
<p>Das ist ein Absatz.</p>
<p>Zeile 1<br>Zeile 2</p>

3. Links & Bilder

  • <a href="...">...</a>: Link
  • <img src="..." alt="...">: Bild
<a href="https://example.com">Zur Webseite</a>
<img src="bild.jpg" alt="Mein Bild">

4. Listen

  • <ul>: Ungeordnete Liste
  • <ol>: Geordnete Liste
  • <li>: Listenelement
<ul>
  <li>Erstes Element</li>
  <li>Zweites Element</li>
</ul>

5. Tabellen

<table>
  <tr>
    <th>Name</th>
    <th>Alter</th>
  </tr>
  <tr>
    <td>Max</td>
    <td>21</td>
  </tr>
</table>

6. Formulare

<form>
  <label>Name:<input type="text" name="name"></label>
  <input type="submit" value="Absenden">
</form>

7. Kommentare & Attribute

  • <!-- Das ist ein Kommentar -->
  • src, href, alt, class, id sind wichtige Attribute.
<img src="logo.png" alt="Logo" width="200">

8. Einbindung von CSS & JavaScript

<link rel="stylesheet" href="style.css">
<script src="script.js"></script>

9. Beispiel: Komplette Mini-Webseite

<!DOCTYPE html>
<html lang="de">
  <head>
    <meta charset="UTF-8">
    <title>Mini-Webseite</title>
  </head>
  <body>
    <h1>Crashkurs HTML</h1>
    <p>HTML ist einfach zu lernen!</p>
    <a href="https://www.w3schools.com">Mehr lernen</a>
  </body>
</html>
