HTML Crash Kurs für Anfänger

Schritt-für-Schritt Anleitung, ausführliche Erklärungen, Kommentare und Tipps

1. Was ist HTML?

HTML steht für HyperText Markup Language und ist das Grundgerüst jeder Webseite. HTML sagt dem Browser, was angezeigt werden soll – z.B. Überschriften, Text, Bilder, Listen oder Links.

Für Einsteiger: HTML ist keine Programmiersprache! Es beschreibt nur die Struktur deiner Inhalte. Man nennt HTML auch „Auszeichnungssprache“.

2. Wie schreibe ich HTML?

  1. Öffne einen einfachen Texteditor (z.B. Notepad, VS Code, Atom).
  2. Speichere deine Datei als index.html.
  3. Schreibe HTML-Code hinein und speichere ab.
  4. Öffne die Datei durch Doppelklick im Browser und sieh dir das Ergebnis an.
Tipp: Mit Rechtsklick > Öffnen mit kannst du die Datei gezielt im Browser starten.

3. Grundstruktur einer HTML-Seite

Jede HTML-Seite braucht einen festen Aufbau. Kommentare sind im Beispiel eingebaut:

<!-- Grundstruktur: Kopiere dies in deine index.html -->
<!DOCTYPE html> <!-- Sagt dem Browser, dass es HTML5 ist -->
<html lang="de">
  <head> <!-- Kopfbereich: hier stehen Metadaten -->
    <meta charset="UTF-8"> <!-- Zeichensatz: Umlaute & Sonderzeichen -->
    <title>Meine erste Webseite</title> <!-- Titel im Browser-Tab -->
  </head>
  <body> <!-- Hier kommt alles, was du sehen willst -->
    
  </body>
</html>

4. Überschriften & Texte

Nutze Überschriften, um Bereiche deiner Seite zu gliedern. <h1> ist die Hauptüberschrift, <h2> bis <h6> sind Unterüberschriften:

<h1>Hallo Welt!</h1>
<h2>Mein erster Abschnitt</h2>
<p>Das ist ein Absatz. Hier kannst du normalen Text schreiben.</p>
<strong>fett</strong> und <em>kursiv</em> <!-- Text hervorheben -->
<br> <!-- Zeilenumbruch -->
Hilfestellung: Überschriften helfen Suchmaschinen und erleichtern die Lesbarkeit. Verwende <h1> nur einmal pro Seite!

5. Bilder & Links

Bilder und Links machen deine Seite lebendig:

<a href="https://example.com">Zur Beispielseite</a> <!-- Link zu einer anderen Seite -->
<img src="bild.jpg" alt="Beschreibung"> <!-- Bild einfügen, alt=Text falls Bild fehlt -->
Hilfestellung: Lege Bilder im gleichen Ordner ab wie deine index.html oder nutze eine Online-URL.

6. Listen

Listen ordnen deine Inhalte:

<ul> <!-- Ungeordnete Liste (Punkte) -->
  <li>Erstes Element</li>
  <li>Zweites Element</li>
</ul>
<ol> <!-- Geordnete Liste (Nummern) -->
  <li>Punkt 1</li>
  <li>Punkt 2</li>
</ol>
Hilfestellung: Listen können auch verschachtelt werden – probiere <ul> innerhalb von <li>.

7. Tabellen

Tabellen sind hilfreich für strukturierte Daten:

<table> <!-- Tabellen-Anfang -->
  <tr> <!-- Tabellenreihe -->
    <th>Name</th> <!-- Überschrift -->
    <th>Alter</th>
  </tr>
  <tr>
    <td>Max</td> <!-- Tabellenzelle -->
    <td>21</td>
  </tr>
</table>
Hinweis: Nutze Tabellen nur für Daten, nicht für Layout! Für Layout und Design ist CSS zuständig.

8. Formulare

Mit Formularen können Besucher Daten eingeben:

<form>
  <label>Name: <input type="text" name="name"></label>
  <input type="submit" value="Absenden">
</form>
Hilfestellung: Ohne weitere Programmierung passiert beim Absenden noch nichts. Für echte Formulare brauchst du später z.B. JavaScript oder PHP.

9. Kommentare und Attribute

Kommentare helfen dir, den Code zu verstehen. Attribute geben Elementen Zusatzinfos:

<!-- Das ist ein Kommentar, er erscheint nicht auf der Webseite -->
<a href="seite.html" target="_blank">Link im neuen Fenster</a> <!-- target sorgt für neues Tab/Fenster -->
<img src="logo.png" alt="Logo" width="150"> <!-- Breite in Pixeln -->
Hilfestellung: Nutze Kommentare großzügig, um dir Notizen zu machen!

10. CSS & JavaScript einbinden

Mit CSS gestaltest du deine Seite, mit JavaScript wird sie interaktiv:

<link rel="stylesheet" href="style.css"> <!-- CSS für Farben, Layout, Schriften -->
<script src="script.js"></script> <!-- JavaScript für Interaktivität -->
Hilfestellung: Lege style.css und script.js im gleichen Ordner ab wie deine index.html.

11. Schritt-für-Schritt Beispiel: Mini-Webseite

  1. Öffne einen Editor und speichere als index.html.
  2. Kopiere folgenden Code hinein:
<!DOCTYPE html>
<html lang="de">
  <head>
    <meta charset="UTF-8">
    <title>Meine erste Webseite</title>
  </head>
  <body>
    <h1>Willkommen!</h1>
    <p>Das ist meine erste eigene HTML-Seite.</p>
    <a href="https://www.w3schools.com">Hier kannst du mehr lernen</a>
    <img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="W3Schools Logo">
    <ul>
      <li>Punkt 1</li>
      <li>Punkt 2</li>
    </ul>
    <!-- Das ist ein Kommentar, nur für dich sichtbar! -->
  </body>
</html>
  1. Speichere ab und öffne die Datei im Browser.
  2. Ändere Texte, füge neue Elemente hinzu – experimentiere!
Hilfestellung: Lerne am besten durch Ausprobieren. HTML ist fehlertolerant – der Browser zeigt bei kleinen Fehlern trotzdem etwas an!
